# Springboot-with-mongoDB

Simple Todo apps with springboot and Mongo DB

[pre required]
1. java SDK min version 11
2. mongo db local server latest
3. code editor (IDE)
4. API test, i use postman

hot to use it :
1. clone repo
2. run mongo db server
3. run java as application
4. test on postman

[Feature]
1. create todo
2. get all todo
3. get todo by id
4. update todo by id
5. delete todo by id
- validator
- exception

Enjoyy
